GOAL
Add and seed a “Top 100 Grower Materials” library into KEBB Ag for Central Valley permanent-crop operations (almonds, pistachios, walnuts, citrus, stone fruit, blueberries). These materials must be usable in:
1. Quick Log
2. Templates
3. Materials & Products library search/filter
4. Future cost engine / unit matching flows

This is frontend-only and should persist using the existing zustand + localStorage pattern.

==================================================
FEATURES TO BUILD
==================================================

1) MATERIAL LIBRARY SEED
Add a one-time seed function that loads the Top 100 materials below into the product/material library if the library is empty.

2) LIBRARY PAGE
Create or extend the existing “Materials & Products” page with:
- search by name / alias
- filter by category
- filter by sub-type
- compact card/list view
- mobile-friendly design
- badge for category
- default unit displayed

3) QUICK LOG INTEGRATION
In Quick Log, add a “Materials Used” section above Notes.
User can:
- add one or more materials
- optionally enter rate
- optionally enter rate unit
- optionally enter total amount
- optionally enter notes per material
- preserve this in the log snapshot

4) TEMPLATE INTEGRATION
Allow templates to include preselected materials and optional default rates/units.

5) SIMPLE TOTALS MATH
If:
- block acres are known
- rate unit contains “/AC”
then compute:
totalNeeded = rate * acres

Examples:
- 15 GAL/AC on 22.5 AC = 337.5 GAL
- 3 LB/AC on 40 AC = 120 LB

If the unit does not contain /AC, do not compute automatically. Just store as entered.

6) HELP PAGE
Under Community & Help, add “Grower Materials Guide”
Include short sections:
- Nutrition
- Soil & Water Amendments
- Fungicides
- Herbicides
- Insecticides & Miticides
- Adjuvants / Oils / Water Helpers
Keep the copy practical and simple.
