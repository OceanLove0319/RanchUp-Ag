# STEP 3 — Active Season Window Without seasonYear
Create file: `src/utils/season.ts`

Implement:
`getActiveSeasonWindow(block): { startISO: string; endISO: string; label: string }`

Rules:
- Determine current Pacific year from todayPacificISO().slice(0,4)
- Determine cropType via inferCropType(block.variety)
- If cropType === STONE_FRUIT:
  - Early:  `${year}-02-01` to `${year}-06-30`
  - Mid:    `${year}-02-01` to `${year}-08-31`
  - Late:   `${year}-02-01` to `${year}-10-31`
  - label:  `Active season (Stone fruit ${block.seasonGroup}): Feb 1 – <end>`
- Else (CITRUS/NUT/OTHER):
  - `${year}-01-01` to `${year}-12-31`
  - label: `Active season: Jan 1 – Dec 31`

IMPORTANT: Do not add seasonYear right now. Use this derived window everywhere Season Spend is computed.
