Build a super-basic MVP “PCA Desk” inside the existing KEBB Ag app.

Goal:
Add a very simple Pest Control Adviser workflow so the app is useful not only for growers, but also for a PCA managing multiple ranches and blocks. This should feel clean, mobile-friendly, and truck-friendly. Keep it lightweight and frontend-only, using the current React + Vite + TypeScript + Tailwind + shadcn/ui stack and existing mock-data / Zustand patterns. Do not introduce a backend.

IMPORTANT:
This is an MVP. Do not overbuild. We only need enough to demonstrate the PCA value clearly in a live demo.
